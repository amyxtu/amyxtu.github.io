Analysis

    #MovePlayerEval

    Evaluation is in pawns from White's perspective (positive favors White). Engine: Stockfish, running client-side via WebAssembly, depth 10. A move is flagged when the evaluation swings more than 1.5 pawns against the player who just moved.

    The full app also turns this into natural-language coaching using the OpenAI API. That step runs server-side (OpenAI doesn't allow direct browser calls), so it's part of the downloadable app, not this in-browser demo.
